# 🛒 NexaMart

NexaMart is a full-stack MERN e-commerce platform that provides a complete online shopping experience with secure authentication, product management, shopping cart functionality, payment integration using Razorpay, and a comprehensive admin dashboard.

## 🚀 Live Demo

**Application:** https://nexamart-2jdb.onrender.com

## ✨ Features

### Customer Features

* User Registration & Login using JWT Authentication
* Browse and search products
* Product detail page
* Add/Remove items from cart
* Update product quantities
* Secure checkout process
* Razorpay payment integration
* Order confirmation page
* User profile with order history

### Admin Features

* Secure admin authentication
* Dashboard with analytics
* Add new products
* Edit existing products
* Delete products
* Manage users
* View and manage orders

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Atharva-0707/NexaMart.git
```

Navigate to the project:

```bash
cd NexaMart
```

Install all dependencies:

```bash
npm run install-all
```

Create a `.env` file inside the **backend** directory and add the following environment variables:

```env
PORT=
MONGO_URI=
JWT_SECRET=

CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=

RAZORPAY_KEY_ID=
RAZORPAY_KEY_SECRET=

FRONTEND_URL=
```

Run the development server:

```bash
npm run dev
```

Build the frontend:

```bash
npm run build
```
